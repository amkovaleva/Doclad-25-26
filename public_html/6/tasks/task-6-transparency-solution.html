Задание 2: Прозрачность (РЕШЕНИЕ)

Пример текста Без прозрачности
Пример текста opacity: 0.5 (весь блок)
Пример текста rgba (только фон)

Что использовано:

  • opacity: 0.5 - делает прозрачным весь элемент (фон + текст)
  • background-color: rgba(255, 0, 0, 0.5) - только фон полупрозрачный
  • Текст в блоке с rgba остаётся чётким и читаемым
  • Текст в блоке с opacity становится бледным
