Задание 2: CSS спрайты

Задача: Создать 4 иконки социальных сетей с помощью CSS спрайта.

Что нужно сделать:

  • Создать 4 блока размером 48x48 пикселей
  • Использовать одну картинку-спрайт (resource/4.png)
  • Показать нужную иконку через background-position
  • Facebook: background-position: 0 0
  • Twitter: background-position: -48px 0
  • YouTube: background-position: -96px 0
  • Instagram: background-position: -144px 0

Подсказка: Используйте background-image и background-position
