🎨 Цвета и фоны в CSS

📌 1. Форматы цветов

HEX: #3498db (Синий)
Текст на синем фоне
RGB: rgb(46, 204, 113) (Зеленый)
Текст на зеленом фоне
RGBA: rgba(52, 152, 219, 0.3)
Прозрачный фон
HSL: hsl(60, 100%, 50%) (Желтый)
Желтый фон
HSLA: hsla(300, 100%, 50%, 0.1)
Прозрачный фиолетовый
Свойство color:
p { color: #333; }

Этот текст написан цветом #333 (темно-серый).

🔍 2. Прозрачность: opacity vs rgba()

opacity (влияет на весь элемент):
opacity: 1
opacity: 1;
opacity: 0.5
opacity: 0.5;
opacity: 0.2
opacity: 0.2;
rgba() (прозрачность только фона):
rgba(231,76,60,0.5) фон
background: rgba(231,76,60,0.5);
Непрозрачный для сравнения

Текст внутри rgba-блока остается черным (непрозрачным), в отличие от opacity.

🖼️ 3. Фоны: цвет, картинка, позиционирование

background-color: #f1c40f;
Просто желтый фон
background-image: pattern1.png;
Фон с картинкой-паттерном

background-repeat:

background-repeat: repeat;
background-repeat: repeat-x;
background-repeat: repeat-y;
background-repeat: no-repeat;

background-size (растягивание фона):

background-size: cover;
Растягивает с сохранением пропорций, заполняя весь элемент
background-size: contain;
Вмещается целиком, могут быть пустые области
background-size: 100% 100%;
Растягивает без сохранения пропорций
background-size: auto (по умолчанию);
Оригинальный размер картинки

background-position:

background-position: center;
background-position: right top;
background-position: 25% 50%;

Универсальное свойство background:

background: оранжевый + паттерн по центру
background: #e67e22 url('pattern2.png') no-repeat center;

🌈 4. Градиенты

Линейные градиенты:

linear-gradient(to right, red, yellow);
linear-gradient(45deg, #000, #fff);
linear-gradient(to bottom right, #3498db, #9b59b6, #e74c3c);

Градиент радуга:

linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);
linear-gradient(90deg, red 0%, orange 15%, yellow 30%, green 50%, blue 70%, indigo 85%, violet 100%);
С указанием процентов для каждого цвета

Радиальные градиенты:

radial-gradient(circle, #f1c40f, #e67e22, #e74c3c);
radial-gradient(circle, red 10%, yellow 30%, green 70%, blue 100%);

Градиент с процентами (цветовые стопы):

linear-gradient(90deg, red 0%, red 20%, yellow 20%, yellow 40%, green 40%, green 60%, blue 60%, blue 80%, purple 80%, purple 100%);
Четкие границы между цветами
linear-gradient(90deg, red 0%, orange 30%, yellow 50%, green 70%, blue 100%);
Плавные переходы с распределением по процентам

🧩 5. CSS Спрайты

Что такое спрайт? Одно большое изображение, содержащее несколько маленьких иконок.

Имитация спрайта (4 в одном):

В одном блоке 4 фоновых изображения с разными позициями.

Реалистичный пример спрайта для соцсетей:

Представьте, что это одна картинка шириной 192px (4 иконки по 48px):

Использование спрайта: Показываем только нужную иконку через background-position.

background-position: 0 0;
Facebook
background-position: -48px 0;
Twitter
background-position: -96px 0;
Instagram
background-position: -144px 0;
YouTube
/* CSS для спрайта */
.sprite-item {
  width: 48px; height: 48px;
  background: url('sprite.png');
}
.sprite-fb { background-position: 0 0; }
.sprite-tw { background-position: -48px 0; }
.sprite-ig { background-position: -96px 0; }
.sprite-yt { background-position: -144px 0; }
