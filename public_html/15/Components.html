Кнопки

Используйте настраиваемые стили кнопок Bootstrap для взаимодействия с формами, диалоговыми окнами, т.п. Имеется поддержка нескольких размеров, состояний и т.д.

Группа кнопок

Сгруппируйте несколько кнопок в одну строку или сложите их в вертикальный столбец. Оберните серию кнопок с помощью .btn в .btn-group.

Для групп кнопок требуется соответствующий атрибут role и явная метка, чтобы вспомогательные технологии, такие как программы чтения с экрана, идентифицировали кнопки как сгруппированные и сообщали о них. Используйте role="group" для групп кнопок или role="toolbar" для панелей инструментов кнопок. Затем используйте aria-label или aria-labelledby, чтобы пометить их.
Кпопки бывают самыми базовыми, могут иметь смешанные или контурные (то есть без фона) стили. Можно объединять чекбоксы и радиокнопоки в единую группу кнопок, создавать паннель инструментов.
Вместо того, чтобы применять классы изменения размера кнопок к каждой кнопке в группе, просто добавьте .btn-group-* к каждой .btn-group, включая каждую при вложении нескольких групп.

Кнопки могут быть выпадающими, могут находиться горизонтально.

Компонент слайд-шоу для цикличного повторения элементов - карусель изображений или текстовых слайдов.
Вы должны добавить класс .active к одному из слайдов, иначе карусель не будет видна. Также не забудьте установить уникальный id в файле .carousel для дополнительных элементов управления, особенно если вы используете несколько каруселей на одной странице. Элементы управления и индикаторы должны иметь атрибут data-bs-target (или href для ссылок), который соответствует id элемента .carousel.

Модальное окно

Используйте модальный JavaScript плагин Bootstrap для добавления диалоговых окнон на ваш сайт для лайтбоксов, уведомлений пользователей или полностью настраиваемого контента.

  • Модальные окна сделаны на HTML, CSS и JavaScript. Они располагаются поверх всего документа. Блокируют прокрутку тела документа, вместо него прокручивая модальные окна.
  • Клик вне модального элемента автоматически закрывает его.
  • Из-за семантики HTML5 атрибут autofocus не работает с модальными элементами Bootstrap

Навигационная панель

Навигационные ссылки на панели навигации основаны на наших параметрах .nav с их собственным классом-модификатором и требуют использования класс переключатель для правильного адаптивного стиля. Навигация в навигационных панелях также будет увеличиваться, чтобы занимать как можно больше горизонтального пространства, чтобы содержимое навигационной панели было надежно выровнено.
Добавьте класс .active в .nav-link, чтобы указать текущую страницу. Обратите внимание, что Вам также следует добавить атрибут aria-current в активный .nav-link.

Нумерация страниц

Мы используем большой блок связанных ссылок для нашей нумерации страниц, что делает ссылки труднодоступными и легко масштабируемыми, обеспечивая при этом большие области попадания. Всплывающие сообщения

Настраиваемые всплывающие сообщения для предупреждающих уведомлений пользователей.
Всплывающие сообщения (Toasts) - это легкие уведомления, разработанные для имитации push-уведомлений, которые были популярны в мобильных и настольных операционных системах. Они построены с flexbox, поэтому их легко выровнять и расположить.
Всплывающие сообщения будут автоматически скрываться, если вы не укажете autohide: false.

Подсказки

Настраиваемые всплывающие подсказки используют CSS3 для анимации, и атрибуты данных для локального хранения заголовков.
Полагаются на стороннюю библиотеку Popper для позиционирования. Вы должны включить popper.min.js перед bootstrap.js или использовать один bootstrap.bundle.min.js, который содержит Popper.
Вы должны инициализировать всплывающие подсказки, прежде чем их можно будет использовать. Один из способов инициализировать все всплывающие подсказки на странице — выбрать их по их атрибуту data-bs-toggle.
