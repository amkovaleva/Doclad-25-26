Что такое формы в Bootstrap?

Формы в Bootstrap — это набор стилей и компонентов для создания удобных, адаптивных и доступных форм. Они включают поля ввода, чекбоксы, радиокнопки, селекты, группы ввода и валидацию. Формы обеспечивают единый вид в разных браузерах и устройствах.

Bootstrap использует классы вроде form-control для стилизации элементов, делая их современными и отзывчивыми.

Обзор форм в Bootstrap

Формы строятся на классах вроде form-control для полей ввода, form-label для меток и form-text для подсказок. Например: 

<form>
  <div class="mb-3">
    <label for="exampleInputEmail1" class="form-label">Адрес электронной почты</label>
    <input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp" />
    <div id="emailHelp" class="form-text">Мы никогда не поделимся вашей почтой.</div>
  </div>
  <button type="submit" class="btn btn-primary">Отправить</button>
</form>
Мы никогда не поделимся вашей почтой.

1. Элементы управления формами (Form Controls)

Класс form-control применяется к <input>, <textarea> и <select>. Поддерживает типы: text, email, password, number, file и т.д. Размеры: form-control-sm, form-control-lg. 

<input type="text" class="form-control" placeholder="Введите текст" />
<textarea class="form-control" rows="3"></textarea>

2. Выпадающие списки (Select)

Класс form-select для <select>. Поддерживает multiple, size, disabled. Размеры: form-select-sm, form-select-lg. 

<select class="form-select">
  <option selected>Выберите...</option>
  <option value="1">Один</option>
</select>

3. Чекбоксы и радиокнопки (Checks and Radios)

Класс form-check для чекбоксов и радиокнопок. Для свитчей — form-switch. Inline: form-check-inline. Без меток: используйте aria-label. 

<div class="form-check">
  <input class="form-check-input" type="checkbox" id="check1" />
  <label class="form-check-label" for="check1">Стандартный чекбокс</label>
</div>

4. Ползунки (Range)

Класс form-range для <input type="range">. Поддерживает min, max, step, disabled. 

<input type="range" class="form-range" min="0" max="5" step="0.5" id="customRange" />
