1. font-size — размер шрифта

Абсолютные единицы (px)

Этот текст 12px

Этот текст 16px (стандартный)

Этот текст 24px

Этот текст 32px

Относительные единицы (rem, em, %)

1.5rem — зависит от корневого элемента (обычно 16px × 1.5 = 24px)

Родитель с font-size: 20px

1.5em = 30px (зависит от родителя)

Родитель 20px

150% = 30px

Ключевые слова

small — маленький

medium — средний

large — большой

x-large — очень большой

2. font-family — выбор шрифта

Шрифты с засечками (serif)

Georgia, Times New Roman — подходит для книг

Шрифты без засечек (sans-serif)

Arial, Helvetica — современный вид

Моноширинные шрифты (monospace)

Courier New, Consolas — как в терминале

Стек шрифтов (несколько вариантов)

Сначала ищет Roboto, потом Arial, потом любой sans-serif

3. font-weight — насыщенность шрифта

100 — Thin (тонкий)

300 — Light (светлый)

400 — Normal (обычный)

500 — Medium (средний)

600 — Semi Bold (полужирный)

700 — Bold (жирный)

900 — Black (очень жирный)

Ключевые слова

normal = 400

bold = 700

bolder — жирнее родителя

lighter — светлее родителя

4. font-style — наклонное начертание

normal — обычный текст

italic — курсив (настоящий)

oblique — наклонный (искусственный)

oblique 10deg — наклон с углом

5. font — сокращённая запись

font: italic 700 20px/1.5 Arial

font: 24px/1.2 Georgia

font: bold 16px/1.4 'Courier New'

Порядок: font-style font-weight font-size/line-height font-family

7. text-align — выравнивание текста

По левому краю (left)

По центру (center)

По правому краю (right)

По ширине (justify). Этот текст растянут так, чтобы строки были одинаковой ширины. Иногда это создаёт большие пробелы между словами, но выглядит аккуратно в книгах.

Текст baseline

9. text-decoration — украшение текста

none — без линии

underline — подчеркивание

overline — надчеркивание

line-through — зачеркивание

underline red — красное подчеркивание

wavy blue — волнистое синее

double green — двойное зеленое

10. text-transform — преобразование регистра

none — как есть

uppercase — все заглавные

LOWERCASE — все строчные

capitalize — каждое слово с большой
