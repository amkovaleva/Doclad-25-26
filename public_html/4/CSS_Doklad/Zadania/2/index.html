Задание 2: Создай кнопку

Время: 10-12 минут

Создай универсальный класс .btn для кнопок. Используй CSS-переменные!

Требования:

  • Класс должен работать для <a>, <button>, <input>
  • Используй переменную --btn-color для цвета фона
  • При наведении (:hover) цвет должен меняться
  • Текст должен быть белым

HTML заготовка:

<a class="btn" href="#">Ссылка-кнопка</a> <button class="btn">Кнопка</button> <input class="btn" type="submit" value="Отправить">
