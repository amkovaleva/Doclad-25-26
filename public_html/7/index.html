Задания для самостоятельной работы

Задание 1

Воспроизведите макет с изображения, используя свойства float и content.

Задание 2

Повторите шаблон с изображения, но на этот раз примените свойство box-sizing: border-box для упрощения расчетов.

Задание 3

Создайте родительский контейнер, внутри которого разместите два элемента с классом box.
Примените к ним свойство float. Затем предотвратите схлопывание высоты родителя, используя различные методы очистки потока (clearfix).
