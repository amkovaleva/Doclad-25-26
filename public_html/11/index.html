Трансформации и анимации на практике

1. Трансформации (transform)

1.1 Перемещение — translate()

Пример: Кнопка "Купить" сдвигается при наведении (эффект нажатия)

Нажимай Купить!
transform: translateY(-3px) — поднимает кнопку при наведении
transform: translateY(0) — возвращает на исходное место при нажатии

1.2 Поворот — rotate()

Пример: Иконка загрузки или стрелка раскрытия меню

▼
Наведи на стрелку
transform: rotate(180deg) — разворачивает стрелку

1.3 Масштабирование — scale()

Пример: Увеличение фото в галерее при наведении

overflow: hidden — скрывает всё, что выходит за рамки
object-fit: cover — картинка заполняет контейнер
transform: scale(1.15) — увеличивает контейнер
position: relative — для позиционирования img

1.4 Наклон — skew()

Пример: Эффект "скорости" для баннера

⚡ СКИДКА 50%
⚡ АКЦИЯ ДНЯ
⚡ РАСПРОДАЖА
Наведи на объект
transform: skewX(-15deg, -5deg) — наклоняет элемент по оси X
transform: skewY(-15deg, -5deg) — наклоняет элемент по оси Y
transform: skew(-15deg, -5deg) — наклоняет элемент по обеим осям

1.5 Комбинирование трансформаций

Пример: Карточка товара с 3D-эффектом

Товар
transform: rotateY(15deg) translateZ(20px) — 3D поворот + сдвиг

📝 Задания для самостоятельной работы

Выполни эти задания за ~10 минут каждое. Ответы не даны — попробуй сам!

Задание 1: Кнопка с эффектом нажатия

Задача:

Сделай кнопку, которая при наведении:

  • Увеличивается в размере (scale)
  • Меняет цвет фона плавно (transition)
  • При нажатии (active) уменьшается — имитация нажатия

Подсказка: Используй :hover и :active псевдоклассы.

Задание 2: Бесконечная загрузка

Задача:

Создай анимированный индикатор загрузки:

  • Круг, который вращается бесконечно
  • Длительность: 1 секунда
  • Линейная функция времени
  • Добавь градиентную границу (border)

Подсказка: Используй @keyframes rotate и border-top-color.

Задача:

Сделай карточки, которые появляются последовательно:

  • Анимация: плавное появление (opacity 0 → 1) + подъём (translateY)
  • Первая карточка появляется сразу
  • Каждая следующая с задержкой на 0.3 секунды
