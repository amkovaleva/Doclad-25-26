Примеры списков с разной нумерацией

1. Маркированный список с вложением

  • Фрукты
    • Яблоки
    • Груши
  • Овощи
    • Огурцы
    • Помидоры

2. Обычная нумерация (по умолчанию)

Стандартный порядок: 1, 2, 3...

  1. Первый этап
  2. Второй этап
  3. Третий этап
  4. Четвертый этап

3. Обратная нумерация (reversed)

Атрибут reversed заставляет список идти в обратном порядке (от 5 до 1). Количество элементов считается автоматически.

  1. Пятый шаг (начало списка)
  2. Четвертый шаг
  3. Третий шаг
  4. Второй шаг
  5. Первый шаг (конец списка)

4. Нумерация со сдвигом (start)

Атрибут start="10" начинает нумерацию не с единицы, а с указанного числа.

  1. Десятый пункт
  2. Одиннадцатый пункт
  3. Двенадцатый пункт

5. Комбинация: Обратная + Старт

Можно начать обратный отсчет с любого числа. Здесь список из 3 элементов начинается с 100.

  1. Сто (начало)
  2. Девяносто девять
  3. Девяносто восемь (конец)
