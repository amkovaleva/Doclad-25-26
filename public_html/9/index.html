Пример 1: Базовая концепция и оси

Разбор HTML тегов для Flex-контейнеров:

  • <header>, <nav>, <main>, <section>, <article>, <footer>:
    • Смысл: Семантические теги, задающие логическую структуру страницы (шапка, навигация, основной контент и т.д.).
    • Закрытие: Все указанные теги являются парными и обязательно должны закрываться соответствующим тегом (напр., </header>).
    • Расположение: Располагаются внутри тега <body> согласно иерархии макета. Например, <header> обычно является первым крупным блоком страницы.
    • Дочерние элементы: Могут содержать любые элементы. В контексте Flexbox — прямые потомки этих тегов автоматически становятся гибкими «флекс-элементами» при активации режима display: flex.
    • Атрибуты: Чаще всего используются глобальные атрибуты: class, id, style.

  • <div>:
    • Смысл: Универсальный блочный контейнер без смысловой нагрузки. Используется, когда семантические теги не подходят или контейнер нужен исключительно для стилизации (напр., группировка карточек в сетку).
    • Закрытие: Обязательно требует закрывающего тега </div>.
    • Расположение: Может находиться в любом месте внутри <body>.
    • Дочерние элементы: Любые HTML-элементы.
    • Атрибуты: class, id, title.

Разбор CSS свойств:

  • display: Определяет тип контейнера. Значения: flex (блочный), inline-flex (строчный).

Пример 2: Свойство flex-direction

Свойство flex-direction: Определяет направление главной оси. Входит в короткую запись flex-flow. Применяется к контейнеру.

  • row: Слева направо (по умолчанию).
  • row-reverse: Справа налево.
  • column: Сверху вниз.
  • column-reverse: Снизу вверх.

Пример 3. Свойство flex-wrap

flex-wrap: Определяет, будут ли элементы переноситься на новую строку.

  • nowrap: В одну линию (по умолчанию).
  • wrap: Перенос на новые строки.
  • wrap-reverse: Перенос в обратном порядке.

Пример 4. Свойство justify-content

justify-content: Выравнивает элементы вдоль главной оси и распределяет свободное пространство.

  • flex-start / flex-end: К началу / концу оси.
  • center: По центру.
  • space-between: Крайние элементы к краям.
  • space-evenly: Одинаковые расстояния между всеми.

Пример 5. Свойство align-items

align-items: Выравнивает элементы по поперечной оси в текущей строке.

  • stretch: Растягивание на всю высоту (по умолчанию).
  • flex-start / flex-end / center: В начало / конец / центр.
  • baseline: По базовой линии текста.
