Пример 1: Базовая концепция и оси

Разбор HTML тегов:

  • <div>: Универсальный блочный элемент для группировки. Обязательно закрывается </div>. Может располагаться в любом месте <body>. Дочерние — любые элементы [2, 4]. Атрибуты: class, id.

Разбор CSS свойств:

  • display: Определяет тип контейнера. Значения: flex (блочный), inline-flex (строчный) [5].
  • flex-direction: Устанавливает главную ось. Значения: row (строка), row-reverse, column (колонка), column-reverse [6].
