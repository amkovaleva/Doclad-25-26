Примеры

Пример 1: Базовая концепция и оси

Разбор HTML тегов:

  • <div>: Универсальный блочный элемент для группировки. Обязательно закрывается </div>. Может располагаться в любом месте <body>. Дочерние — любые элементы. Атрибуты: class, id.

Разбор CSS свойств:

  • display: Определяет тип контейнера. Значения: flex (блочный), inline-flex (строчный).
  • flex-direction: Устанавливает главную ось. Значения: row (строка), row-reverse, column (колонка), column-reverse.

Пример 2: Выравнивание и распределение (justify-content)

Разбор CSS свойств:

  • justify-content: Выравнивает элементы по главной оси.
  • Значения: flex-start (в начало), flex-end (в конец), center (центр), space-between (крайние к краям), space-around (равные отступы вокруг), space-evenly (равные расстояния везде).

Пример 3: Гибкость элементов (flex-grow)

Разбор CSS свойств:

  • flex: Краткая запись для flex-grow, flex-shrink и flex-basis. По умолчанию: 0 1 auto.
  • flex-grow: Коэффициент роста. 0 — не растет, 1+ — заполняет свободное место.
